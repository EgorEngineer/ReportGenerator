# Report Generator
Генератор отчетов по новым адресным объектам из Федеральной информационной адресной системы (ФИАС).

## Что делает программа
1. Подключается к API ФИАС
2. Скачивает последний пакет изменений
3. Обрабатывает адресные объекты
4. Формирует отчеты в двух форматах

### Требования
- .NET 6.0 SDK или выше ([скачать](https://dotnet.microsoft.com/download))
- Подключение к интернету

### Запуск программы

**Windows:**
1. Дважды кликните на файл `run.bat`
2. Дождитесь завершения работы
3. Отчеты откроются автоматически

**Альтернативный способ:**
1. Открыть терминал в папке проекта
2. Ввести следующую команду:
```bash
dotnet run
```

## Результат работы

Программа создаст папку `FiasReports` с двумя отчетами:
- `report_{дата составления отчёта}.html` - для просмотра в браузере
- `report_{дата составления отчёта}.docx` - для просмотра в Word

## Структура отчета

Отчет содержит:
- Дату изменений
- Общее количество объектов
- Таблицы по уровням (города, улицы, и т.д.)
- Сортировку по наименованию

## Возможные проблемы

**"ОШИБКА: .NET SDK не установлен"**
- Установите .NET 6.0 или новее с официального сайта

**"Response status code does not indicate success"**
- Проверьте подключение к интернету
- Возможны временные проблемы с сервером ФИАС
